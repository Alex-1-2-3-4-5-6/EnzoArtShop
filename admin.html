2 Upload to GitHub

After generating the JSON above, follow these steps to get it live:

  1. Go to your EnzoArtShop GitHub repo
  2. Click Add file → Upload files and upload your photo into the images/ folder (create it if it doesn't exist by typing images/ before the filename in the upload box)
  3. Go back to the repo, click data.json, then click the pencil (edit) icon
  4. Select all the text in the file and paste in the new JSON you copied above
  5. Click Commit changes — your artwork goes live in about 1 minute!
